# Step 2.3 - Injection Queue, Local Queue and Work Stealing

## Metadata

- Status: Planned
- Depends on: Step 2.2
- Estimate: 2 days

## Goal

建立外部注入队列、Worker 本地队列、Steal 和 Sleep/Wake，不要求首版无锁。

## Responsibility

- External submit 进入 Injection Queue。
- Worker 子任务优先进入本地 LIFO。
- 空闲 Worker 从其他 Worker FIFO steal。

## Path Plan

- `JobQueue.hpp/.cpp`
- `WorkerContext.hpp/.cpp`
- `JobScheduler.cpp`
- `tests/unit/JobQueueTests.cpp`
- `tests/stress/JobStealingStressTests.cpp`

## Implementation

- 有界 Mutex Deque 作为正确性版本。
- round-robin/victim seed 可复现。
- Semaphore/condition variable 睡眠，提交时避免丢失 wakeup。
- 统计 local pop、injection pop、steal success/fail、sleep。

## Unit Tests

- 外部提交和 Worker 递归提交都完成。
- 多 Worker 发生可观测 Steal。
- 空闲时 Worker 睡眠而非 busy spin。
- Submit 与 sleep 竞态不丢任务。
- 100 万任务压力测试无重复/丢失。

## Delivery Checklist

- [ ] 队列竞争可被 Profiler Counter 观察。
- [ ] Worker 可配置固定随机种子用于测试。
- [ ] 无锁优化被明确延期。
- [ ] 单 Worker 模式可用。

## Boundaries

- 不实现依赖和 Wait。
- 不使用 detached thread。
- 不在队列中保存裸 Job 指针。
