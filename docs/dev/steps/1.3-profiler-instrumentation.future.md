# Step 1.3 - Instrumentation API and RAII Zones

## Metadata

- Status: Planned
- Depends on: Step 1.2
- Estimate: 1 day

## Goal

提供可编译关闭的 Scope、Function、Counter、Instant 和 Frame Mark API。

## Responsibility

- 宏只负责构造轻量 API 调用。
- RAII Scope 保证异常和提前 return 时 Zone 闭合。
- Profiler Session 决定是否记录。

## Path Plan

- `Profiler.hpp`
- `ProfilerScope.hpp`
- `ProfilerMacros.hpp`
- `ProfilerSession.cpp`
- `tests/unit/ProfilerInstrumentationTests.cpp`

## Implementation

- `CHIKA_PROFILE_SCOPE`、`FUNCTION`、`COUNTER`、`INSTANT`、`FRAME`。
- Compile definition `CHIKA_ENABLE_PROFILING`。
- Disabled 宏不求值可选昂贵 Payload。
- Zone Begin/End 使用相同 Name ID 和线程身份。

## Unit Tests

- 嵌套 Zone 顺序正确。
- early return 和异常仍产生 End。
- Disabled build 不写事件。
- Counter 支持整数和浮点值。

- [ ] 测试不依赖真实时间精度。
