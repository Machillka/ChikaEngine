## Goal

提供可编译关闭的 Scope、Function、Counter、Instant 和 Frame Mark API。

## Responsibility

- 宏只负责构造轻量 API 调用。
- RAII Scope 保证异常和提前 return 时 Zone 闭合。
- Profiler Session 决定是否记录。

## Implementation Record

- `ProfilerSession` 统一管理运行时开关和帧边界；`ProfilerScope`/`ProfilerFrameScope` 用 RAII 保证提前返回和异常路径闭合。
- 提供 Scope、Function、Counter、Instant、Frame 宏；名称 ID 缓存在函数局部 static，编译关闭后宏不求值参数。

## Changed Files and Reasons

- `ProfilerSession`、`ProfilerScope`、`ProfilerMacros`、`Profiler.hpp`：形成业务模块唯一公开入口。
- `ProfilerDisabledMacroTests.cpp`：用独立 translation unit 验证 `CHIKA_ENABLE_PROFILING=0` 的参数不求值。

## Knowledge and Design Notes

- 编译期开关删除代码，运行时开关保留可随时启停的观测能力；二者解决不同发布需求。
- RAII 只保证词法作用域配对，跨线程 begin/end 不属于该 API 的合法用法。
