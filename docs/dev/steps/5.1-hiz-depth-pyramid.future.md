# Step 5.1 - Hi-Z Depth Pyramid Foundation

## Metadata

- Status: Optional / Planned
- Depends on: Step 4.8 and measured occlusion bottleneck
- Estimate: 2-3 days

## Goal

生成可跨帧读取的 Depth Pyramid，并正确处理 depth convention、mip、resize 和 camera cut。

## Responsibility

- RenderGraph 表达 depth source、mip chain 和跨帧资源。
- Compute Shader 只做保守 downsample。
- Renderer 管理 history validity。

## Path Plan

- `Assets/Shaders/hiz_downsample.comp`
- `Render/include/ChikaEngine/DepthPyramid.hpp`
- `Render/src/DepthPyramid.cpp`
- `RenderPipelinePasses.cpp`
- `tests/unit/DepthPyramidTests.cpp`

## Implementation

- 明确 normal/reversed-Z 和 conservative reduce 规则。
- 创建完整 mip chain Texture/View。
- 每 mip Dispatch 与 subresource barrier。
- resize、camera cut、scene switch 后 history invalid。

## Unit Tests

- mip count/extent 覆盖奇数尺寸和 1x1。
- reduce rule 对 normal/reversed-Z 正确。
- RenderGraph subresource lifetime/barrier 正确。
- history invalid 时不被 consumer 读取。

- [ ] Texture View 有 Debug Name。

## Boundaries
